Features:
- users can choose from 6 avatars provided
- Users can post questions either anonymously or not.
- Every time a user posts a question, they can receive a streak.
- Streaks can be used to give other users a token for providing the best answer to their question.
- Users can choose a best answer even if only one user answered their post, or among multiple replies if many users responded.
- Once a best answer is selected, users cannot select another best answer, even if additional answers are added.
- Tokens can be converted into money, and users can cash them out (this feature is not developed yet, but is planned for future implementation).
